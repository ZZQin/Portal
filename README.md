# PORTAL Website

Premium garage door e-commerce website built with HTML, CSS, and JavaScript.

## Structure

- `index.html` - Homepage
- `products.html` - Product catalog
- `about.html` - About Us
- `contact.html` - Contact page
- `tier-*.html` - Tier detail pages (ONE, CRAFT, LEGACY)
- `product-*.html` - Individual product pages (9 products)
- `images/` - Product images

## Brand

Following PORTAL Brand Guidelines with:
- Portal Black (#1A1A1A)
- Cloud White (#F5F5F5)
- Sky Blue (#4A90E2) - ONE tier
- Forge Gold (#D4A574) - CRAFT tier
- Navy Depth (#1E3A5F) - LEGACY tier

## Local Development

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000
